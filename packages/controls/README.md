# @fml/controls

## What are they?

Controls are pieces of forms that are responsible for tracking state of the
form's data.

There are three general classifications of controls:

1. Fields, which store a single value. Examples include text inputs, checkboxes
    and date pickers.
2. Models, which store data for an object. Each property of the object has a
    corresponding control of the appropriate data type for that property.
3. Lists, which store a list of the same data type.
